## 폴더 구조

Next.js 9버전부터 pages 디렉토리가 src 디렉토리 하위에서 역할을 할 수 있게 되었다.
그렇기 때문에 익숙한 리액트 구조와 동일한 방식으로 src 디렉토리 아래에서 대부분의 파일을 관리하는 형식으로 폴더 구조를 설정하였다.
<a href='https://nextjs.org/blog/next-9-1#src-directory-support'>Nextjs 개발팀 공식문서</a>
